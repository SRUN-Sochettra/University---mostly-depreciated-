# University Coursework & Projects

## Overview
This repository serves as a centralized monorepo for academic coursework, lab assignments, and project implementations. The codebase spans multiple disciplines including full-stack web development, cross-platform mobile development, and advanced data structure architecture.

## Repository Structure

### `/Advanced Data Structures with Python`
Python-based implementations and Jupyter Notebooks focusing on algorithmic efficiency and data organization.
* **Data Structures:** Binary Search Trees (BST), Singly Linked Lists, Maps.
* **Projects:** LRU Cache implementation (`lru_cache_project.ipynb`), performance testing metrics.

### `/flutter projects`
Cross-platform mobile application development utilizing Dart and the Flutter framework.
* **`nham24/`**: Application interface clone.
* **`semester2/`**: State management modules and fundamental app structures (`counter_app.dart`, `theme_logic.dart`).
* **`flutter_application_1/`**: Basic UI/UX module testing.

### `/web`
Frontend web development directory focusing on responsive design and modern CSS frameworks.
* **`tailwindCLI/`**: Advanced styling using Tailwind CSS, including a functional Spotify web interface clone (`spotify-clone/`).
* **`week1/` & `schoolwork/`**: Core HTML5, CSS3 layout exercises, CSS flexbox implementations, and UI debugging.
* **Assignments:** Form validations, custom layouts, and interactive DOM manipulation via JavaScript.

### `/lg lg`
Miscellaneous Python algorithmic scripts and execution files (`algorithm.py`).

## Tech Stack
* **Languages:** Python, Dart, JavaScript, HTML5, CSS3
* **Frameworks & Libraries:** Flutter, React, Tailwind CSS
* **Tools:** Jupyter Notebook, Node.js
